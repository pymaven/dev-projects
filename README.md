# dev-projects

## 📺 K-Clips (Open Korean Video Corpus)

An open-source, collaborative repository designed for language instructors to curate, timestamp, and share authentic video clips for Korean language classrooms. 

### 🚀 Features
* **Progressive Filtering:** Multi-tier dynamic sidebar filtering by Proficiency Level, Lesson, and Grammar Point.
* **Instructor Collaboration:** Instantly updates via a shared cloud spreadsheet without requiring redeployments or Git commits.
* **Classroom Optimized:** High-impact, sandboxed video container layout built using privacy-enhanced embedding to bypass live authentication blocks.
* **Interactive Scaffolding:** Built-in accordion layout for target Korean transcripts and English translations.
